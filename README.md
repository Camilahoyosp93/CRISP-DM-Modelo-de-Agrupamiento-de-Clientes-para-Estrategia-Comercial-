## Descripción
Se implementó la metodología **CRISP-DM** para segmentar 
8,950 clientes de tarjetas de crédito y diseñar estrategias 
comerciales diferenciadas para la oferta de una segunda tarjeta.

## Objetivo
Determinar si segmentar los clientes en 4 grupos es razonable 
según las métricas de desempeño
